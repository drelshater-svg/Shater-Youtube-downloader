🎛️ Shater-Youtube-downloader

🎛️ حزمة استوديو متكاملة لتحميل الفيديوهات وتخصيصها ومعالجتها
أداة ذات واجهة ويب محلية لتحميل الفيديوهات، ودمج قنوات الصوت والفيديو عالية الدقة (Adaptive High-Res)، ودمج العلامات المائية النصية المخصصة، ومزج مسارات الصوت الخلفية المتكررة (Loops)، وحرق الترجمات والنصوص المنسقة على الفيديو


📋 المتطلبات الأساسية (إلزامي)
يستخدم هذا التطبيق برمجية FFmpeg في جميع عمليات معالجة الصوت والفيديو الأساسية. يجب أن تكون البرمجية مثبتة على نظام تشغيلك:
    • نظام Windows: قم بتحميل حزمة (Essentials compilation package) ثم أضف مسار مجلد /bin الخاص بها إلى متغيرات البيئة في النظام Path.
    • نظام macOS: قم بالتثبيت عبر حزمة Homebrew باستخدام الأمر: brew install ffmpeg
    • نظام Linux: قم بالتثبيت عبر حزمة apt باستخدام الأمر: sudo apt install ffmpeg


🚀 تعليمات البدء السريع
    1. تحميل المستودع: قم بعمل Clone للمشروع أو تحميله كملف مضغوط zip على قرصك المحلي.
    2. التأكد من وجود مجلدات الملفات: تأكد من وجود مجلدين باسم music و fonts داخل الدليل الرئيسي للمشروع. ضع ملفاتك المخصصة بصيغة .mp3 أو .ttf بداخلهما.
    3. تشغيل تطبيق الويب:
        ◦ 🪟 نظام Windows: ما عليك سوى النقر المزدوج على ملف السكربت run.bat.
        ◦ 🍏 نظام macOS و 🐧 نظام Linux: افتح واجهة السطر البرمجي (Terminal) داخل مجلد المشروع ونفّذ الأمر التالي:



     ```bash
     chmod +x run.sh
     ./run.sh
     ```


سيقوم السكربت تلقائياً بإنشاء بيئة عمل معزولة (Sandbox) باستخدام مدير الحزم السريع uv، وجلب طبقة إعدادات لغة Python 3.12 المحسّنة، وتثبيت الحزم المطلوبة، ثم فتح لوحة التحكم المحلية على متصفحك الخاص فوراً!

⚠️ الأغراض التعليمية وإخلاء المسؤولية القانونية
تم إنشاء هذا المشروع بدوافع تعليمية ولأغراض الأرشفة الشخصية فقط. لقد صُمم كأداة تجريبية محلية لدراسة الرسوم البيانية لفلاتر الفيديو (Video filter graphs)، والدمج الآلي للوسائط المتعددة عبر مسارات برمجية FFmpeg، وتطبيق واجهات المستخدم المحلية باستخدام Python و Gradio.
    • إخلاء المسؤولية: يوفر المؤلف هذا البرنامج "كما هو" (As is) بموجب رخصة MIT، ولا يتحمل مطلقاً أي مسؤولية قانونية أو أدبية عن كيفية تفاعل المستخدمين النهائيين مع هذه الشفرة البرمجية، أو أي فقدان للبيانات، أو عدم استقرار قد يحدث للنظام.
    • الامتثال لشروط الخدمة: إن تحميل الوسائط المحمية بحقوق الطبع والنشر دون إذن صريح من مالكها أو خارج النطاق الذي توفره منصات البث بشكل رسمي يعتبر انتهاكاً مباشراً لشروط خدمة YouTube.
    • مسؤولية المستخدم: تقع المسؤولية الكاملة على عاتق المستخدم النهائي في التحقق من الوضع القانوني، وحقوق الطبع والنشر، وأذونات التوزيع لأي روابط بث يتم معالجتها من خلال هذه الأداة الشخصية. يُمنع استخدام هذا البرنامج في التوزيع غير المصرح به للمواد الإعلامية أو التطبيقات التجارية العامة.
